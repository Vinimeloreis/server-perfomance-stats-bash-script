# Server Perfomance Stats
A basic bash script to analyze server perfomance stats
## Getting Started
1. **Clone the repo**
   
   ```
   git clone https://github.com/Vinimeloreis/server-perfomance-stats-bash-script.git
3. **Navigate to the file folder**
   
   ```
   cd server-perfomance-stats 
5. **Run bash command**
   
   ```
   bash server-stats.sh 
> [!NOTE]
> bash command doesn't need execute permission to run file just make sure you have read permission so the bash can read the commands in the file
</br>

> If you don't know how to check for permissions click <a href="https://askubuntu.com/questions/528411/how-do-you-view-file-permissions">here</a>
